# 核心功能

## 计时系统

| 功能 | 说明 |
|------|------|
| 预设时长 | 30秒 / 60秒 / 90秒 / 2分钟 |
| 自定义时长 | 10秒 - 600秒可调 |
| 倒计时显示 | 大字号居中，最后10秒变红警示 |
| 暂停/继续 | 随时暂停训练，保留进度 |
| 重置 | 一键恢复初始状态 |

## 暂停统计

训练过程中记录暂停数据：
- **暂停次数** — 点击暂停的次数
- **暂停时长** — 每次暂停的时长（秒）
- **总暂停时长** — 所有暂停累加
- **完成状态** — 无暂停显示「完美完成 🎯」，有暂停显示「暂停 X 次，共 Y 秒」

## 呼吸引导

呼吸节奏跟随 4-2-4 模式：
- **吸气** (4秒) — 屏幕圆环放大 + 文字提示「吸气...」 + 振动反馈
- **屏息** (2秒) — 保持状态 + 文字提示「屏息...」 + 振动反馈
- **呼气** (4秒) — 圆环缩回 + 文字提示「呼气...」 + 振动反馈
- **循环** — 自动进入下一个呼吸周期
- **振动反馈** — 配合呼吸节奏振动，闭眼也能感知当前阶段

## 音频提示

| 时机 | 音效 |
|------|------|
| 最后10秒 | 每秒「滴」声提示 |
| 30秒姿态检查 | 提示音 + 随机姿态提醒 |
| 每30秒（训练≥60秒） | 鼓励提示音 + 文字鼓励 |
| 训练完成 | 三音阶成功音效 |

## 姿态检查

## 数据统计

本地存储以下数据：
- **今日完成次数** — 每日重置
- **本周完成次数** — 按周统计
- **累计训练时长** — 实际训练时长（扣除暂停）

历史记录最多保存 100 条训练记录，包含：
- 日期时间、训练模式、设定时长
- 暂停次数、总暂停时长
- 完成状态标签

## 历史记录面板

点击底部「总时长」区域查看：
- 最近训练列表（按时间倒序）
- 每条记录显示：日期时间、模式、时长、暂停情况
- 完美完成的记录显示「完美」标签

### 趋势图表

历史面板新增「趋势」标签页，展示多折线图：
- **横坐标**：训练日期
- **纵坐标（左轴）**：训练次数
- **纵坐标（右轴）**：训练时长（秒）、休息时长（秒）

三条折线分别展示：
- 绿色：每日训练次数
- 蓝色：每日累计训练时长
- 红色：每日累计休息时长

帮助用户追踪训练习惯，激励坚持每日训练。

## 云端同步 (Supabase)

**架构**: 离线优先 + 云端同步

| 特性 | 说明 |
|------|------|
| 匿名认证 | Supabase 匿名登录，用户无感知 |
| 离线优先 | localStorage 为主存储，断网可正常使用 |
| 云端同步 | 训练完成后自动同步到 Supabase |
| 冲突处理 | localStorage 优先，本地数据不丢失 |
| 跨设备 | 同一用户在不同设备可看到累计统计 |

**同步机制**:
- 启动时：恢复/创建匿名会话 → 同步待同步记录 → 拉取云端统计
- 训练完成：写入 localStorage → 异步写入 Supabase
- 网络恢复：自动同步待同步记录 → 合并云端统计

**数据表**:
- `profiles` — 用户资料（匿名用户自动创建）
- `sessions` — 训练记录
- `user_stats` — 用户统计聚合
- `leaderboard` — 排行榜

**部署**: Vercel 静态托管 + Supabase 后端即服务

## 个人中心 / 设置

点击右上角 ⚙️ 按钮打开设置页面：

### 个人 Tab
- 头像显示（默认 👤）
- 昵称编辑（本地 + 云端同步）
- 训练统计（总次数、累计时长）
- 账户信息（邮箱/登录提示）

### 设置 Tab
- **语言选择**: 简体中文 / English / 粤语
- **语音播报开关**: 启用/禁用语音指导
- **播报语言**: 中文 / English / 粤语
- **退出登录**: 邮箱用户可退出
