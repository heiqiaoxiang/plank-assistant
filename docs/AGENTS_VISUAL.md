# AGENTS.md 可视化版

## Project Work Habits (Plank Assistant)

---

### 📋 Documentation Sync — FEATURES.md 同步规则

| 场景 | 操作 |
|------|------|
| 新增功能 | 添加到对应模块章节 |
| 功能改进 | 更新现有条目的说明/状态 |
| 功能废弃 | 标记为 deprecated 或移除 |

**目的**：确保功能列表始终反映当前实现状态。

---

### 🔄 Feature Testing — Playwright 测试流程

```
┌─────────────────────────────────────────────────────────────┐
│                      测试工作流                               │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   ┌─────────┐    ┌─────────┐    ┌─────────────┐            │
│   │ 1.运行  │───▶│ 2.截图  │───▶│ 3.保存报告  │            │
│   │ E2E测试 │    │ 关键场景 │    │ report.html │            │
│   └─────────┘    └─────────┘    └─────────────┘            │
│                                                             │
│              保存到 test-results/screenshots/               │
│              命名: {feature-name}-{scenario}.png            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**关键场景**：
- 初始状态 (initial)
- 交互状态 (interaction)
- 完成状态 (completion)

### ✅ Commit Message Convention — 提交信息规范

**原则**：像 release notes 一样写，关注「解决了什么」而非「改了什么」

| 风格 | 示例 |
|------|------|
| ❌ Bad | "Modified app.js timer logic" |
| ✅ Good | "Fixed timer not pausing correctly when user taps pause button" |
| ❌ Bad | "Updated breathRing animation" |
| ✅ Good | "Fixed breath ring animation stuttering during transition" |
