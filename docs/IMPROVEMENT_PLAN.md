# 平板支撑助手 - 系统性改进计划

> 生成日期：2026-03-31  
> 核心方向：不依赖 AI 姿态检测，打造极致沉浸式平板支撑体验

---

## 一、产品定位重构

**核心差异化方向**：`最专注、最沉浸的平板支撑体验`

| 维度 | 现状 | 目标 |
|-----|------|------|
| 训练模式 | 4种，但登山者是假的 | 3个真实模式 or 删除登山者 |
| 呼吸引导 | 纯视觉动画 | 视觉 + 振动 + 声音三位一体 |
| 激励系统 | 排行榜隐藏太深 | 社交证明立即可见 |
| 文案风格 | Emoji 表情包风格 | 克制简洁的专业风格 |

---

## 二、功能优先级

### P0 - 立即执行

- [ ] **1. 去掉 Emoji 文案**
  - 完成庆祝语：`"燃烧中"` / `"完成"` / `"坚持"` 
  - 鼓励提示：`"继续"` / `"稳住"` / `"加油"`
  - 原则：每个画面只出现一个关键词

- [ ] **2. 加入振动反馈 (Haptic Feedback)**
  ```javascript
  // 呼吸引导时配合振动
  navigator.vibrate?.([200, 100, 200]); // 吸气
  navigator.vibrate?.([200]);           // 屏息
  navigator.vibrate?.([100, 100, 200]); // 呼气
  ```
  - 用户闭眼时也能跟上呼吸节奏
  - 需适配 iOS Safari 的 Vibration API

- [ ] **3. 排行榜前置**
  ```
  当前：统计区点击 → 历史 → 切换Tab → 登录 → 排行榜 (5步)
  优化：底部Tab直接显示"本周排名"+"Top 3头像" (1步)
  ```
  - 在历史/排行榜按钮旁显示简短排名摘要
  - 未登录用户显示"登录查看排名"入口

- [ ] **4. 登山者模式决策**
  - [ ] 选项A：做真动态训练（检测左右腿交替抬高）
  - [ ] 选项B：直接删除该模式（减少用户困惑）

### P1 - 下个版本

- [ ] **5. 声音设计升级**
  - Ambient sound：轻柔的背景音（森林/海浪/雨声）
  - 呼吸引导时轻微提示音
  - 用户可选择开启/关闭

- [ ] **6. 训练计划/课程**
  - 7天入门挑战
  - 30天进阶计划
  - 每日推荐训练组合

- [ ] **7. 数据可视化**
  - 周报：本周 vs 上周对比曲线
  - 月度热力图：哪几天训练最多
  - 进步曲线：坚持时长的变化

### P2 - 可选功能

- [ ] **8. Apple Watch / 手环同步**
- [ ] **9. iOS Widget 小组件**
- [ ] **10. 冥想模式延伸**

## 三、视觉语言升级

### 3.1 呼吸环动效优化
- 优化贝塞尔曲线，让呼吸节奏更自然
- 加入微弱的光晕效果（glow）
- 颜色随呼吸周期微妙变化

### 3.2 完成庆祝重构
- 去掉 emoji 堆砌
- 用更精致的粒子系统
- 核心信息放大：`"2分30秒"` + `"坚持"` 或 `"完成"`

### 3.3 字体规范
- 数字用等宽字体（tabular-nums）保持稳定感
- 主标题保持最小字号，确保可读性
