# 核心功能

## 计时系统

| 功能 | 说明 |
|------|------|
| 预设时长 | 30秒 / 60秒 / 90秒 / 2分钟 |
| 自定义时长 | 10秒 - 600秒可调 |
| 倒计时显示 | 大字号居中，最后10秒变红警示 |
| 暂停/继续 | 随时暂停训练，保留进度 |
| 重置 | 一键恢复初始状态 |

## 暂停统计

训练过程中记录暂停数据：
- **暂停次数** — 点击暂停的次数
- **暂停时长** — 每次暂停的时长（秒）
- **总暂停时长** — 所有暂停累加
- **完成状态** — 无暂停显示「完美完成 🎯」，有暂停显示「暂停 X 次，共 Y 秒」

## 呼吸引导

呼吸节奏跟随 4-2-4 模式：
- **吸气** (4秒) — 屏幕圆环放大 + 文字提示「吸气...」
- **屏息** (2秒) — 保持状态 + 文字提示「屏息...」
- **呼气** (4秒) — 圆环缩回 + 文字提示「呼气...」
- **循环** — 自动进入下一个呼吸周期

## 音频提示

| 时机 | 音效 |
|------|------|
| 最后10秒 | 每秒「滴」声提示 |
| 30秒姿态检查 | 提示音 + 随机姿态提醒 |
| 每30秒（训练≥60秒） | 鼓励提示音 + 文字鼓励 |
| 训练完成 | 三音阶成功音效 |

## 姿态检查

每30秒随机显示一条姿态提示：
- 检查：臀部位置
- 检查：肩部下沉
- 检查：核心收紧
- 检查：不要塌腰
- 检查：呼吸节奏

## 鼓励提示

训练时长 ≥ 60秒时，每30秒显示一条鼓励：
- 坚持住！
- 加油！
- 保持姿势！
- 你很棒！
- 继续！

## 训练模式

| 模式 | 图标 | 说明 |
|------|------|------|
| 经典平板支撑 | 🧘 | 标准姿势 |
| 左侧平板支撑 | 🙃 | 锻炼左侧核心 |
| 右侧平板支撑 | 🤪 | 锻炼右侧核心 |
| 登山者 | 🏔️ | 动态组合 |

## 完成庆祝

训练结束时触发：
- 全屏庆祝动画覆盖层
- 30个彩色粒子向外爆炸
- 显示完成时长 + 暂停统计 + 随机鼓励文案
- 成功后音效（Do-Re-Mi 三音阶）

鼓励文案库：
- 「太棒了！继续保持！」
- 「燃脂成功！💪」
- 「今天的你比昨天更强大！」
- 「核心力量+1！」
- 「健身打卡完成！」
- 「汗水不会骗人！」

## 数据统计

本地存储以下数据：
- **今日完成次数** — 每日重置
- **本周完成次数** — 按周统计
- **累计训练时长** — 实际训练时长（扣除暂停）

历史记录最多保存100条训练记录，包含：
- 日期时间、训练模式、设定时长
- 暂停次数、总暂停时长
- 完成状态标签

## 历史记录面板

点击底部「总时长」区域查看：
- 最近训练列表（按时间倒序）
- 每条记录显示：日期时间、模式、时长、暂停情况
- 完美完成的记录显示「完美」标签

## PWA 支持

- 可添加到 iOS/Android 主屏幕
- 离线可用（缓存所有资源）
- 全屏沉浸式体验
- 深色主题适配系统
