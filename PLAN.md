# Plank - 平板支撑助手

## 1. Concept & Vision

一款极简但专业的平板支撑计时器，专为手机端优化。核心体验是「沉浸式」——锻炼时只显示计时和必要提示，消灭一切干扰。带有呼吸节奏引导和完成后的成就感动画。

## 2. Features

### 核心功能
- **单次计时**: 默认60秒倒计时，可自定义 30s/60s/90s/120s/自定义
- **呼吸引导**: 屏幕随呼吸节奏微微起伏 (吸气4s → 屏息2s → 呼气4s)
- **间歇提示**: 每30秒有音频提示「加油」
- **完成庆祝**: 结束时粒子爆炸 + 鼓励文案

### 进阶功能
- **多训练模式**: 
  - 经典平板支撑
  - 侧平板支撑 (左右各计时)
  - 登山者组合
- **历史记录**: 本地存储，展示本周统计数据
- **表单检查**: 每30秒显示「臀部位置」检查点

## 3. Technical Approach

- **单页 HTML + CSS + Vanilla JS**: 无框架依赖，极速加载
- **PWA**: 可添加到主屏幕，离线可用
- **LocalStorage**: 存储设置和历史
- **Web Audio API**: 精准的间歇提示音
- **CSS 动画 + requestAnimationFrame**: 流畅的呼吸视觉效果

## 4. UI/UX Design

- **深色主题**: 减少眼睛疲劳，专注锻炼
- **大字计时**: 中心 40% 显示剩余时间
- **最小化按钮**: 仅「开始」「暂停」「重置」
- **触屏友好**: 最小点击区域 48x48px

### 视觉风格
```
背景: #0a0a0a (深黑)
主色调: #00d4aa (薄荷绿 - 代表活力)
强调色: #ff6b6b (珊瑚红 - 完成时)
文字: #ffffff / #888888
```
